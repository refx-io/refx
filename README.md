## About the project
Welcome RefX into your hiring toolkit! Leveraging Zero-Knowledge technology, With RefX, each verification is immutable and trusted, providing you with peace of mind throughout the hiring process. Plus, our platform is incredibly efficient, allowing you to access reference information anywhere, anytime, with just a few clicks.

## Getting Started

First, run the development server:

```bash
npm run dev
# or
yarn dev
# or
pnpm dev
# or
bun dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

You can start editing the page by modifying `app/page.js`. The page auto-updates as you edit the file.

This project uses [`next/font`](https://nextjs.org/docs/basic-features/font-optimization) to automatically optimize and load Inter, a custom Google Font.
